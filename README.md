This tool encrypts speech for conversations
made by (SX)

 to install 

     git clone https://github.com/Sx123-mpa/speech_encryptor.git
     
 cd file

     cd speech_encryptor

how to use

(to open menu) :--

    python3 speech_encryptor.py -h

(Encryption) :--

    python3 speech_encryptor.py -e (test)
 
(To decrypt) :-- 

    python3 speech_encryptor.py -d (test)
